# ☁️ Raspberry Pi Personal Cloud

## 🚀 Project Overview
This project aims to build a personal cloud system using a Raspberry Pi.

The goal is to create a secure and accessible storage solution similar to Google Drive.

---

## 🎯 Objectives
- Store files locally
- Access files from anywhere
- Secure connection using VPN (WireGuard)

---

## 🛠️ Technologies
- Raspberry Pi
- Linux (Raspberry Pi OS)
- Nextcloud (planned)
- WireGuard VPN (planned)

---

## 📡 Features (planned)
- File storage
- Remote access
- Secure VPN connection
- Multi-device access (PC / phone)
